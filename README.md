# Emoji-Web

## Deploy react App to github page
- intstall Node and npm
- install yarn package

```
npm install -g yarn
```
- install github page packages in your devDependencies.
```
yarn add -D gh-pages
```
or 
```
npm install gh-pages --save--dev
```

- modify package.json

```
"homepage":"https://52147.github.io/Emoji-Web/"
```
